# calcula_acao_judi
Sistema para cálculo de diferenças de reajuste salarial, incluindo verbas como gratificação, adicional noturno e outras parcelas remuneratórias.  Realiza a apuração automática dos reflexos em 13º salário e férias, aplicando índices de atualização monetária conforme período informado, garantindo precisão nos valores corrigidos.
