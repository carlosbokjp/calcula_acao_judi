# calcula_acao_judi
Sistema desenvolvido para cálculo de diferenças de reajuste salarial, contemplando verbas como:
Gratificação
Adicional Noturno
Horas Extras
Outras parcelas remuneratórias
O sistema também realiza automaticamente os reflexos sobre 13º salário e férias, aplicando índices de atualização monetária conforme o período informado.
Funcionalidades
Cálculo de diferenças mensais
Aplicação de percentual de reajuste
Cálculo automático de reflexos em:
13º salário
Férias + 1/3 constitucional
Atualização monetária por período
Integração com API de índices (ex: INPC, IPCA, TR)
Apuração detalhada por evento
Regra de Cálculo
O sistema:
Identifica o percentual de reajuste aplicado no período.
Calcula a diferença sobre cada verba remuneratória.
Aplica os reflexos proporcionais em 13º salário e férias.
Atualiza monetariamente os valores conforme índice selecionado.
Tecnologias Utilizadas
HTML
CSS
JavaScript
API de Índices Econômicos

📌 Objetivo
Automatizar o cálculo de diferenças salariais e reflexos trabalhistas, garantindo precisão, transparência e conformidade com critérios de atualização monetária
